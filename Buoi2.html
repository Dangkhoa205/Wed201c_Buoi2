- ' font-weight: bodl ' : là dùng bôi đậm cho đoạn văn
- ' font - size: 40px ' : tăng kích thước của chữ
- ' font-family: 'Courier New', Courier, monospace: đổi font chữ '
- ' font-style: .... : là các thuộc tính về (in nghiêng,...)'
- text-align: (right, left, center,..): để chỉnh vị trí của đoạn vănvăn
